Installation:
1. Clone repository

     git clone https://github.com/c0n0I/CMPE131ProjectMilestone1.git](https://github.com/c0n0I/CMPE131ProjectMilestone2.git 
   
     cd "CMPE131 Project Milestone1"
3. Create and activate virtual environment
   
     .venv\Scripts\Activate.ps1
5. Install dependencies
   
     pip install -r requirements.txt

Run the App:
1. Start the server
   
     python run.py
3. Open in browser
   
     http://127.0.0.1:5000/

4. Default user is created in database to test the application

     Enter "testuser" in username to login
